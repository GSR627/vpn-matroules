# VPN Matroules

Android VPN Client Application - A lightweight VPN client for Android

## Features

- ✅ Simple and intuitive UI
- ✅ Add multiple VPN configurations
- ✅ One-click connect/disconnect
- ✅ Real-time traffic statistics
- ✅ Support for Shadowsocks protocol
- ✅ Material Design 3 UI

## Requirements

- Android 7.0+ (API 24)
- Android Studio 4.0 or higher
- Kotlin 1.8.0+

## Installation

### Using Android Studio

1. Clone the repository
2. Open in Android Studio
3. Build → Build Bundle(s) / APK(s) → Build APK(s)
4. Install on your device

## Usage

1. **Add Configuration**: Click "Add" button and enter VPN server details
2. **Select Configuration**: Tap on a configuration to select it
3. **Connect**: Click "Connect" button to establish VPN connection
4. **Disconnect**: Click "Disconnect" to stop VPN

## Permissions

- INTERNET: For VPN connection
- CHANGE_NETWORK_STATE: To establish VPN
- BIND_VPN_SERVICE: Required for VPN service
- ACCESS_NETWORK_STATE: To monitor network status
